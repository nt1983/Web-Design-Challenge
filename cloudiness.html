Cloudiness Visualization

The purpose of this project was to analyze weather changes moving closer to the equator. This was accomplished by analyzing data from the OpenWeatherMap API, creating a dataset on more than 500 cities.
After assembling the dataset, we used Matplotlib to pilot Various aspects of weather vs. latitude. Factors we looked at included: temraturare, cloudiness, wind speedand humidity. This site provides the source data and visualization created as part of the analysis as well as explanations and descriptions os any trends and correlations witnessed.
